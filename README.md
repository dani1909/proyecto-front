# proyecto-front

# 📚 Talento Tech Ecommerce

Proyecto de Ecommerce desarrollado como parte de la práctica de desarrollo web frontend.  
La aplicación simula una tienda online de libros con diseño responsive, formulario de contacto y secciones promocionales.

---

## 📱 Responsive Design

El proyecto utiliza Media Queries para adaptarse a distintos tamaños de pantalla:

| Breakpoint | Dispositivo |
|---|---|
| `1200px` | Desktop grande |
| `992px` | Notebook / Tablet horizontal |
| `768px` | Tablet |
| `576px` | Mobile |
